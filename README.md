# ✍️ AI-Based Personality Prediction from Handwriting

## 📌 Project Overview

AI-Based Personality Prediction from Handwriting is a computer vision and machine learning project that analyzes handwriting images to predict personality traits. The system extracts handwriting features such as slant, spacing, pressure, baseline alignment, margins, and letter size, and uses machine learning models to estimate personality traits based on handwriting patterns.

This project combines Computer Vision, Machine Learning, and Artificial Intelligence into a complete end-to-end application with a Streamlit web interface.

---

## 🚀 Features

- 🖼️ Handwriting image upload
- 🧹 Image preprocessing using OpenCV
- ✏️ Handwriting feature extraction
- 🤖 Personality prediction model
- 📊 Personality traits visualization
- 📄 Personality report generation (PDF)
- 🌐 Streamlit web application
- 🧩 Modular project architecture

---

## 🧠 Personality Traits Predicted

The system predicts personality traits based on the Big Five Personality Model:

- Openness
- Conscientiousness
- Extroversion
- Agreeableness
- Neuroticism
